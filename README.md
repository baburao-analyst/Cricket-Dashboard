# 🏏 Dynamic Cricket Performance Analytics Dashboard

![Dashboard Screenshot](Dashboard%20Screenshot.png)

> An interactive Power BI dashboard that provides comprehensive batting and bowling performance analysis for cricket players using dynamic DAX measures, slicers, and advanced data visualization.

---

## 📌 Project Overview

This project is a dynamic and fully interactive Power BI dashboard developed to analyze cricket player performance across multiple formats and years.

The dashboard enables users to seamlessly switch between **Batting** and **Bowling** statistics while automatically updating every KPI card and chart.

Designed with a modern UI and dynamic calculations, this project demonstrates practical business intelligence, data modeling, and dashboard design skills.

---

## 🚀 Key Features

### 🎯 Dynamic Role Switching

- Batting Analysis
- Bowling Analysis

All KPI cards and charts update automatically.

---

### 📊 Interactive Filters

- 👤 Player Selection
- 🏏 Role Selection
- 📅 Year Range
- 🏆 Match Format
  - IPL
  - ODI
  - T20I
  - Test

---

### 📈 Dynamic KPI Cards

Batting

- Innings
- Runs
- Average
- Strike Rate
- Highest Score
- 50's / 100's

Bowling

- Innings
- Wickets
- Average
- Economy
- Best Bowling in Innings (BBI)
- Five-Wicket Hauls

---

### 📉 Interactive Visualizations

- Boundary Distribution
- Strike Rate & Average by Year
- Best Score / BBI by Year
- Runs / Wickets by Year
- Balls by Year
- Dot Percentage by Year

---

### 🔄 Reset Filters Button

Restore the dashboard to its default view with a single click.

---

### 🖼 Dynamic Player Images

Player images automatically update when a different player is selected.
